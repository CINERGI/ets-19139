OGC Web Testing Facility

This free "self-service" testing facility can be used by any developer as often as they like to test their implementations.

This site is created to test metadata instances, as part of the NSF project Community Inventory of EarthCube Resources for Geosciences Interoperability (CINERGI). It is based on the OGC Web Testing Facility.
